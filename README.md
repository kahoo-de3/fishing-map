# 釣りマップ 関東（東京・千葉・神奈川）

スマホ対応の沿岸釣りマップ。衛星写真の上に「水深の色分け・等深線・かけあがり（急斜面）・底質」を重ねて表示し、タップした地点の水深・底質と「狙える魚種（推定）」を表示します。すべて無料データで構築。

## 魚種推定について

タップ地点の「水深＋周辺2kmの底質＋かけあがり＋海域（東京湾奥/東京湾/外海）＋月」から、
関東の定番ターゲット約20種（シロギス・マゴチ・ヒラメ・カサゴ・メバル・クロダイ・アオリイカ・
タチウオ・アマダイ等）をルールベースで絞り込み、今が旬の魚に◎を付けて表示します。
釣果データベースではなく推定です。魚種の追加・条件調整は `docs/index.html` の `FISH` 配列を編集。

## 使い方（ローカル確認）

```
python -m http.server 8765 --directory docs
```

→ ブラウザで http://localhost:8765 を開く。

## スマホで使う（公開手順）

`docs/` フォルダを静的ホスティングに置くだけです。おすすめは GitHub Pages：

1. GitHubに新リポジトリを作成（例: fishing-map）
2. `docs/` の中身をリポジトリ直下にpush
3. Settings → Pages → Branch: main / root を選択
4. 発行されたURLをスマホのSafari/Chromeで開き、「ホーム画面に追加」→ アプリのように使える（GPSの現在地ボタンはHTTPS必須なので公開後に有効になります）

## 構成

- `docs/index.html` — アプリ本体（MapLibre GL JS）
- `docs/data/depth_only.png` — 水深色分け＋等深線オーバーレイ
- `docs/data/slope.png` — かけあがり（急斜面）オーバーレイ
- `docs/data/depth.bin` — タップ水深表示用グリッド（Int16、0.1m単位、32767=陸）
- `docs/data/meta.json` — グリッドの範囲情報
- `docs/data/sediment.json` — 底質ポイント（岩・礫・砂・泥・貝殻・さんご・溶岩、約7,300点）
- `tools/download_etopo.py` — 水深データのダウンロード（NOAA ETOPO 2022, 15秒メッシュ）
- `tools/process.py` — オーバーレイ画像・グリッド生成
- `tools/fetch_sediment.py` — 底質データの取得（海しるAPI。ポータル公開のお試しキー使用。
  お試しキーは予告なく変わることがあるので、失敗したら https://portal.msil.go.jp/howtouse で
  最新キーを確認するか、無料の個別キーを申請して差し替える。
  ※「泥・粘土」のAPIスラッグは公式の綴りミスで `mud-caly`）

## データと精度の注意

- 水深: NOAA **ETOPO 2022**（GEBCO由来、約450mメッシュ）。無料・出典表示のみで利用可。
- 港の中や岸から数百mの細かい根・かけあがりは解像度的に表現できません（有料の海底地形デジタルデータM7000で置き換え可能な設計です）。
- 底質（岩/礫/砂/泥/貝殻など）: 海しるAPI（海上保安庁）の電子海図由来ポイントデータ。
  海図に記載のある地点のみの「点」情報なので、面的な塗り分けではありません。
  ズーム9.5以上で点表示、ズーム12以上で文字ラベル表示。タップすると水深と一緒に底質も表示。
- 表示範囲: 東経138.9〜141.0度、北緯34.82〜35.80度（小田原〜三浦〜東京湾〜房総〜銚子）

## 対象エリアを変える

`tools/download_etopo.py` の `BBOX` を変更 → `download_etopo.py` → `process.py` の順に実行。
（緯度30度未満/45度以上になる場合はETOPOタイル名 N45E135 も変更が必要）
